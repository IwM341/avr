Detailed Description

template<typename size_type, typename guard_type = no_guard>
struct aos::pipe< size_type, guard_type >

class i/o for reading and writing bytes

Template Parameters
size_typeType of buffer size: uint8_t if buffer length < 256 and short otherwise
guard_typetype to preserve pipe from race condition, should have functions lock() and unlock()
